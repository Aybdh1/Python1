CRM-система для магазина

Описание
Учебное десктопное CRM-приложение на Python для управления клиентами, товарами и заказами.
Использует SQLite для хранения данных и Tkinter для интерфейса.
Поддерживает импорт/экспорт (CSV, JSON) и простую аналитику (pandas, matplotlib, seaborn, networkx).

Функционал

Управление клиентами и товарами (добавление, удаление).

Создание заказов с привязкой клиент → товар.

Сортировка заказов по дате или стоимости.

Импорт/экспорт данных (CSV, JSON).

Аналитика: топ-5 клиентов, динамика заказов, граф связей.

Установка зависимостей

Рекомендуется использовать виртуальное окружение:

python -m venv venv
# Windows
venv\Scripts\activate
# macOS / Linux
source venv/bin/activate

pip install pandas matplotlib seaborn networkx
